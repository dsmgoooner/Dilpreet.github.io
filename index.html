Projects

Sales Analysis

In this project, Python, along with several libraries, was employed to conduct a comprehensive analysis of sales data from the year 2019. The dataset encompasses crucial information such as Order ID, Product, Quantity Ordered, Price Each, Order Date, and Purchase Address. By leveraging these data points, the objective was to unearth actionable insights that could refine sales strategies, optimize inventory management, and elevate customer satisfaction.

The primary objectives of this sales analysis are twofold. Firstly, to identify sales trends, we will determine the monthly sales patterns to understand peak sales periods. Secondly, we will delve into product performance, evaluating which products are the popular and contribute the most to revenue. Additionally, we will conduct geographical analysis to analyze sales distribution across different cities to identify key markets. Lastly, we will explore customer purchase behavior, focusing on common product combinations to enhance cross-selling strategies. Throughout the analysis, essential sales metrics such as total revenue, average order value, and product-specific performance will be calculated.For further details and code implementation, please refer to the @Sales Analysis

Yelp Analysis

In the competitive restaurant world, knowing what makes a business successful is super important. This project dives into the Telp dataset to figure out how stuff like reviews, tips, and check-ins from customers affect how well a restaurant does. The dataset, a part of Yelp, covers 8 big areas in the USA and Canada and includes info from four main types of files: business, review, tip, and check-in, all originally in JSON format. To make sense of all this data, we built a @Database and then analyse using both SQL and Python.

I used a bunch of libraries like numpy, pandas, matplotlib, seaborn, datetime, sqlite3, geopy, and IPython.display to dig into the data and analyze it. With the help of SQLite, which links Python and SQL, I managed the data smoothly.For further details and code implementation, please refer to the @Yelp Analysis

Sales Dashboard

This project involves creating an interactive sales dashboard using Tableau, incorporating data from three files: customer, product, and location. The goal is to build a dashboard to help stakeholders, including sales managers and executives, analyze sales performance . The sales dashboard aims to present an overview of sales metrics and trends, allowing for an analysis of year-over-year performanc e and an understanding of sales trends.

Key Features

The dashboard displays a summary of total sales, profits, and quantities for the current and previous year. It presents monthly data for each KPI, highlighting months with the highest and lowest sales. A comparison of sales performance by product subcategories for both years, including a sales and profit comparison, is featured. Additionally, weekly sales and profit data for the current year is shown, with average weekly values and highlights of weeks performing above and below average.For further details and code implementation, please refer to the @Sales Dashboard

Football Prediction

This project delves into the application of machine learning algorithms to predict the outcomes of football matches, leveraging historical match data. Using a dataset that includes various performance metrics and match outcomes, we built and compared three different models: Support Vector Machine (SVM), Random Forest, and Extreme Gradient Boosting (XGBoost). Each model underwent extensive data preprocessing, including one-hot encoding for categorical variables and downsampling for class balance. We then evaluated the models' performance using metrics such as accuracy, precision, recall, and F1-score. The results were visualized using confusion matrices and bar plots, providing a comprehensive view of each model's predictive capabilities.

To further enhance the models, we employed hyperparameter optimization techniques like GridSearchCV and RandomizedSearchCV. These optimization processes identified the best parameters for each algorithm, improving their accuracy and robustness. The optimized Random Forest model achieved the highest accuracy at 67.5%, closely followed by the XGBoost model. The final project not only highlights the strengths of different machine learning models in sports analytics but also demonstrates the importance of data preprocessing and hyperparameter tuning in developing effective predictive models. This project serves as a testament to the potential of machine learning in sports forecasting, offering valuable insights for teams, analysts, and enthusiasts.For further details and code implementation, please refer to the @Football Prediction
